Animación JavaScript HTML DOM

Código de animación

  • Grupo 8
  • Diciembre, 2021
  • Todas las animaciones deben ser relativas a un elemento contenedor.
  • Las animaciones de JavaScript se realizan programando cambios graduales en el estilo de un elemento.


    • El código básico es:

    id = setInterval(frame, 5);

    function frame() {
        if (/* test for finished */) {
        clearInterval(id);
       } else {
        /* codigo para cambiar el elemento style*/
       }
    }
