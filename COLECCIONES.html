Ejemplos del objeto HTML Collection

El getElementsByTagName() método devuelve un HTMLCollection objeto

Ejemplo
const myCollection = document.getElementsByTagName("p");
Se puede acceder a los elementos de la colección mediante un número de índice.
Para acceder al segundo elemento < p> puede escribir
myCollection[1]

Ejemplo al ejecutarlo

Nota: el índice comienza en 0.

Longitud de la colección HTML HTMLC

La lengthpropiedad define el número de elementos en un HTMLCollection:


Ejemplo
myCollection.lengthpropiedad
Ejemplo al ejecutarlo

La length propiedad es útil cuando desea recorrer los elementos de una colección:

Ejemplo
Cambie el color del texto de todos los elementos < p>:
const myCollection = document.getElementsByTagName("p");
for (let i = 0; i < myCollection.length; i++) {
  myCollection[i].style.color = "red";
}
